# CHASED Website

This is the project repository for the CHASED e-commerce website.

## Project Structure
- `index.html`: The main simple landing page.
- `index_old.html`: The previous full version of the website (Backup).
- `assets/`: Directory for images and other static assets.
- `styles/`: CSS files.
- `scripts/`: JavaScript files.

## Local Development
To run the website locally:
1. Open the folder in your terminal.
2. Run `npx serve .` (if you have Node.js installed) or simply open `index.html` in your browser.
